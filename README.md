# Convite de Casamento - Landing Page

Esta é uma landing page de convite de casamento criada com [Next.js](https://nextjs.org) e [Tailwind CSS](https://tailwindcss.com).

## Funcionalidades

- Design elegante com tema dourado e creme.
- Molduras decorativas usando CSS.
- Fontes manuscritas e serifadas para um toque clássico.
- Totalmente responsivo.

## Como Iniciar

Instale as dependências (se necessário):

```bash
npm install
```

Rode o servidor de desenvolvimento:

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## Personalização

- **Nomes e Datas**: Edite `src/app/page.tsx`.
- **Estilos e Cores**: As variáveis de cores e fontes estão em `src/app/globals.css`.
